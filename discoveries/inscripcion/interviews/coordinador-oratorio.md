# Entrevista — Coordinador del Oratorio Vacacional

> Notas crudas de la conversación. Entrevista anonimizada. Se conserva el dolor de negocio relacionado con la coordinación de grupos, edades, comunicación por WhatsApp y seguimiento operativo de las inscripciones.

**Entrevistador:** ¿Cuál es su responsabilidad principal dentro del proceso de inscripción?

**C. (Coordinador del Oratorio Vacacional):** Mi parte tiene que ver con organizar los grupos y asegurar que cada niño quede donde corresponde. También se coordina el tema de WhatsApp, porque cada grupo necesita recibir su enlace o estar vinculado al grupo correcto.

**Entrevistador:** ¿Qué problema genera la clasificación por edades?

**C.:** Cuando se trabaja con varios formularios por edad, se presta para errores. Si el padre llena el enlace equivocado o si alguien revisa rápido, el niño puede quedar mal clasificado. Después toca moverlo o corregir la información.

**Entrevistador:** ¿Cómo debería resolver eso el sistema?

**C.:** El sistema debería calcular o tomar la edad ingresada y asignar automáticamente el grupo correspondiente. Así no dependemos de que el padre elija el enlace correcto ni de que el receptor revise manualmente cada edad.

**Entrevistador:** ¿Los grupos por edad son siempre iguales?

**C.:** No necesariamente. Pueden variar según la organización del año. Por eso sería útil que se puedan configurar los rangos de edad y los grupos. Lo importante es que el sistema use esa configuración para clasificar a los niños.

**Entrevistador:** ¿Qué problema tienen actualmente con los grupos de WhatsApp?

**C.:** El problema es enviar el enlace correcto. Como hay grupos por edad, después de la inscripción hay que ver a qué grupo corresponde cada niño y mandar el link al representante. Eso puede demorarse o equivocarse.

**Entrevistador:** ¿Qué espera que haga el sistema respecto a WhatsApp?

**C.:** Al finalizar la inscripción, el número del representante debería quedar vinculado al grupo correspondiente. Idealmente el sistema lo agregaría automáticamente al grupo de WhatsApp según la edad del niño. Si no se puede, al menos debería enviar de inmediato el enlace correcto.

**Entrevistador:** ¿Cuándo debería enviarse el enlace de WhatsApp?

**C.:** Depende del pago. Si el pago todavía está pendiente de verificación, no siempre conviene enviar el enlace final. Una vez confirmado el pago, el sistema debería habilitar el envío automático del enlace correspondiente.

**Entrevistador:** ¿Qué ocurre si una familia tiene varios niños en grupos diferentes?

**C.:** Ahí el sistema debería identificar cada niño por separado. Puede ser que un representante tenga dos hijos y cada uno pertenezca a un grupo distinto. El mensaje debe ser claro y decir qué niño va a qué grupo.

**Entrevistador:** ¿Cómo se relaciona su trabajo con el receptor de inscripciones?

**C.:** Dependemos de que el receptor registre bien la edad, fecha de nacimiento y datos del niño. Si eso entra mal, la coordinación también sale mal. Por eso la clasificación automática ayuda, pero también debe haber forma de revisar.

**Entrevistador:** ¿Cómo se relaciona su trabajo con Tesorería?

**C.:** Tesorería confirma el pago. Si el sistema usa el estado de pago para enviar el enlace de WhatsApp, entonces necesitamos que ese estado esté actualizado. Coordinación no debería estar preguntando manualmente si ya pagó o no.

**Entrevistador:** ¿Qué información necesita ver para coordinar mejor?

**C.:** Necesito ver niños por grupo, edad, representante, número de WhatsApp, estado de pago, estado de documento y si ya se envió o no el enlace del grupo. Eso nos permite dar seguimiento.

**Entrevistador:** ¿Cuál es el mayor dolor de su rol?

**C.:** La coordinación manual. Revisar edades, pasar nombres a grupos, mandar enlaces y corregir errores consume tiempo. Además, si se envía un enlace equivocado, se genera confusión con las familias.

**Entrevistador:** ¿Cómo sabría que el sistema funciona bien para coordinación?

**C.:** Si los niños quedan clasificados automáticamente, si los grupos se manejan según edades configuradas, y si el enlace de WhatsApp se envía al representante correcto cuando corresponde. Eso reduciría bastante el trabajo manual.
