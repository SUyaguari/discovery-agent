# Entrevista — Tesorera del Oratorio Vacacional

> Notas crudas de la conversación. Entrevista anonimizada. Se conserva el dolor de negocio relacionado con pagos, comprobantes, verificación y constancias de inscripción.

**Entrevistador:** ¿Cuál es su responsabilidad dentro del proceso de inscripción?

**T. (Tesorera del Oratorio Vacacional):** Mi responsabilidad es revisar los pagos. Pueden ser pagos en efectivo o por transferencia, entonces hay que controlar quién pagó, cuánto pagó y si el comprobante realmente fue verificado.

**Entrevistador:** ¿Qué problema existe actualmente con los pagos?

**T.:** El problema es que el proceso puede continuar, pero no siempre queda claro si el pago ya está confirmado. Cuando es transferencia, primero hay que validar el comprobante. Entonces se necesita un estado que diga que el pago está pendiente de verificación.

**Entrevistador:** ¿Qué estados de pago considera necesarios?

**T.:** Como mínimo, pendiente, pago pendiente de verificación y pago verificado. Si el comprobante no corresponde, también debería poder marcarse como rechazado o con observación. Lo importante es que no se confunda un pago enviado con un pago ya verificado.

**Entrevistador:** ¿La inscripción debería detenerse si el pago no está verificado?

**T.:** No necesariamente. El sistema debería permitir continuar con el proceso de inscripción mientras se valida el comprobante. Pero debe quedar claro que el pago todavía no está confirmado.

**Entrevistador:** ¿Qué debería pasar cuando el pago ya está confirmado?

**T.:** Cuando se confirma, el estado cambia a pago verificado. Después de eso se puede habilitar el envío automático del enlace de WhatsApp correspondiente y también dejar el comprobante como válido.

**Entrevistador:** ¿Qué información necesita para verificar una transferencia?

**T.:** Necesito ver el representante, los niños inscritos, el monto, la fecha, el comprobante y algún dato que permita comparar si realmente ingresó el dinero. También se debe saber quién hizo la verificación.

**Entrevistador:** ¿Y cómo se debería manejar el pago en efectivo?

**T.:** Si el pago es en efectivo, quien recibe puede marcarlo como pagado o registrado. Pero igual debe quedar constancia: fecha, monto, responsable y familia inscrita. Eso evita confusiones después.

**Entrevistador:** ¿Qué problema genera no tener comprobante final?

**T.:** Que después las familias preguntan o reclaman y no hay una constancia clara. Un comprobante ayuda tanto a la familia como al Oratorio, porque deja evidencia de la inscripción realizada y del estado del pago.

**Entrevistador:** ¿Qué debe incluir el comprobante de inscripción?

**T.:** Debe incluir fecha de inscripción, datos del representante, nombres de todos los niños inscritos, estado del pago y, si aplica, el grupo o información relacionada. También sería útil un código o número de inscripción.

**Entrevistador:** ¿El comprobante debería enviarse por WhatsApp?

**T.:** Sí, porque es el canal que más usan las familias. Al finalizar el registro, el sistema debería enviar el comprobante automáticamente al WhatsApp del representante. Si el pago todavía está pendiente, el comprobante debe decirlo claramente.

**Entrevistador:** ¿Cómo se relaciona su trabajo con el receptor de inscripciones?

**T.:** El receptor registra la información inicial. Si los datos de la familia o de los niños están incompletos, Tesorería no puede validar correctamente. Por eso el sistema debe mantener todo ordenado.

**Entrevistador:** ¿Cómo se relaciona su trabajo con el coordinador?

**T.:** Coordinación necesita saber si el pago ya está verificado para enviar el enlace de WhatsApp. Entonces el estado de pago debe estar actualizado para que el coordinador no tenga que preguntar manualmente.

**Entrevistador:** ¿Cuál es el mayor dolor de su rol?

**T.:** Verificar pagos sin un estado claro. Si el sistema no diferencia entre pendiente, enviado y verificado, se puede enviar información antes de tiempo o registrar mal una inscripción.

**Entrevistador:** ¿Cómo sabría que el sistema sí mejora el control de pagos?

**T.:** Si puedo revisar pagos pendientes, confirmar transferencias, registrar efectivo, cambiar estados y generar comprobantes claros para las familias. Eso daría más orden y reduciría reclamos.
